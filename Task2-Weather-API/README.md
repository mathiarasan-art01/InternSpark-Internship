# 🌦 Weather API Integration

## 📌 Project Overview

This application fetches real-time weather information using a public API and displays weather conditions for any city entered by the user.

---

## 🚀 Features

- Search Weather by City
- API Integration
- JSON Parsing
- Error Handling
- Real-Time Data

---

## 🛠 Technologies Used

- Python 3
- Requests Library
- JSON
- REST API

---

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python weather_app.py
```
